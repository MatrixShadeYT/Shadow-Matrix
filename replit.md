# Shadow Matrix - MSYT

## Overview
A canvas-based JavaScript game project. The project is built with vanilla HTML, CSS, and JavaScript - no build tools or package managers needed.

## Project Structure
- `index.html` - Main HTML page with embedded CSS styling
- `main.js` - JavaScript canvas animation logic
- `README.md` - Project documentation about creating a Shadow AI chatbot

## Running the Project
The project is served using Python's built-in HTTP server on port 5000.

## Deployment
Configured for static deployment, serving files directly from the root directory.
